Безопасность

Эта страница фиксирует требования, которые должны быть включены перед коммерческим запуском TrafficMind AI.

Production auth

  • Личный кабинет: email/password или magic link, восстановление доступа, защита сессий.
  • Админка: отдельная роль владельца, MFA, audit log и запрет публичного доступа.
  • API: rate limit, CORS whitelist, CSRF-защита для cookie-сессий.

Интеграции

  • OAuth-токены хранить только в зашифрованном token vault.
  • Подключения должны иметь отзыв доступа и журнал синхронизаций.
  • Ошибки внешних API не должны раскрывать секреты пользователю.

Мониторинг

Для релиза нужны error tracking, health checks, uptime alerts, логирование платежей, логирование OAuth и ежедневная проверка фоновых задач.
